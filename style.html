Pullquotes


This is a Pull Quote which floats on the left.

This is just a simple paragraph of text that is placed after the Left Pull Quote. As you can see, this will wrap around the Pull Quote. These are useful for drawing users attention to an image or quote within the text.

This is a Pull Quote which floats on the right.


You may have to adjust the font size of the pull quotes and balance it with the margins to get this to appear balanced. You can alway set a fixed width to the pull quotes as well, or they will simply wrap around the content.

 

 

DropCaps


Drop caps is this large letter D at the start of a paragraph. As you can see, this simple effect can draw much more attention to itself.

This dropcap is styling the letter T to simply be larger. Keep in mind that images can also be used here.
